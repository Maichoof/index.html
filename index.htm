Languages

  • HTML: stands for HyperText Markup Language, and is used to give a webpage base.
  • CSS: stands for Cascading Style Sheets, and is used to customize the style of HTML elements.



    • HTML Elements

    <h1-h6>

    text headings on a webpage. h1 is the largest heading; h6 is the smallest.

    <p>

    used for non-heading text

    <a>

    short for anchor and used to add links to other webpages. Anchor elements typically have an href attribute which is used for hyperlinks.

    <img>

    used to add images to your webpage. Images are self-closing and do not require a closing tag.

    <video>

    used to add videos to your webpage. It uses multiple attributes and a source element.

    <ul>

    used to create lists on your webpage. It requires li elements inside an unordered list.

    <ol>

    used to created numbered lists on your webpage. It requires li elements inside an ordered list.

    <div>

    used to organise HTML elements into different groups, which can be given a class attribute then customized to look significantly better.

    <pre>

    used to preserve its unusual formatting such as large gaps between words or to skip a few lines.


    Web Concepts

    parent/child elements

    used to describe HTML elements that enclose or are enclosed by other elements. For example, the <ul> (unordered list) is the parent and the <li> (list items) are children.

    <html>...</html>

    The root of the HTML document and parent of all other HTML elements on the webpage.

    <head>...</head>

    Enclose other metadata about the site, such as its title.

    <title>...</title>

    Contains the site's title, which is one way users can find your site through a search engine, like Google.
