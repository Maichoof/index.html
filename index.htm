Languages

  • HTML: stands for HyperText Markup Language, and is used to give a webpage base.
  • CSS: stands for Cascading Style Sheets, and is used to customize the style of HTML elements; controls the appearance of content by changing text size, image size, and other attributes.
  • JavaScript: Code that adds interactivity and animation to webpages. JavaScript also detects browser events such as mouse clicks, validates user input such as text entries, and retrieves data from external websites.
  • Ruby: An open-source programming language best known for use in web programming. For example, you want to make a website that takes credit card payments. Ruby can check to make sure the credit card number appears to be valid.
  • Python: An open-source programming language used on the web, in scientific applications, and for data analysis.



    • HTML Elements

    <h1-h6>

    text headings on a webpage. h1 is the largest heading; h6 is the smallest.

    <p>

    used for non-heading text

    <a>

    short for anchor and used to add links to other webpages. Anchor elements typically have an href attribute which is used for hyperlinks.

    <img>

    used to add images to your webpage. Images are self-closing and do not require a closing tag.

    <video>

    used to add videos to your webpage. It uses multiple attributes and a source element.

    <ul>

    used to create lists on your webpage. It requires li elements inside an unordered list.

    <ol>

    used to created numbered lists on your webpage. It requires li elements inside an ordered list.

    <div>

    used to organise HTML elements into different groups, which can be given a class attribute then customized to look significantly better.

    <pre>

    used to preserve its unusual formatting such as large gaps between words or to skip a few lines.

    <marquee>

    something about marquee

    <nav>

    used to organize site navigation menus on a webpage.


    Web Concepts

    parent/child elements

    used to describe HTML elements that enclose or are enclosed by other elements. For example, the <ul> (unordered list) is the parent and the <li> (list items) are children.

    <html>...</html>

    the root of the HTML document and parent of all other HTML elements on the webpage.

    <head>...</head>

    enclose other metadata about the site, such as its title.

    <title>...</title>

    contains the site's title, which is one way users can find your site through a search engine, like Google.

    rem

    represents the default font size for the web browser.

    pixel (px)

    standard unit of measurement for sizing fonts and other HTML elements.

    CSS Selectors

    element

    examplep {} selects <p></p>

    class

    used to target classes of HTML elements; ideal for times when you need to apply the same style to multiple elements.

    example.red{} selects <p class="red"></p>

    id

    used to style an HTML element which has an id attribute; is how you can uniquely identify an element. Think of an id attribute as being like an element's name.

    declaration

    are inside the curly braces and follow a selector. Each declaration must end with a semicolon (;).

    property

    shows what should be modified. It customises the color, width, location of an element. It must be followed by a colon (:).

    value

    shows how the property should be customised.
